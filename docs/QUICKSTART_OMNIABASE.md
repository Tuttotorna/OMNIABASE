# Quickstart OMNIABASE

This document gives the fastest path to seeing OMNIABASE as the representation foundation of the ecosystem.

The public mental model is:

    representation -> base shift -> observation -> invariant residue

---

## Clean install

    git clone https://github.com/Tuttotorna/OMNIABASE.git
    cd OMNIABASE
    python -m pip install -e .
    pytest

---

## What to look for

After installation and tests, look for the smallest available example.

The point is not to prove a full theory.

The point is to identify:

    What object is being observed?
    What representation is used?
    What base or representation shift is applied?
    What feature is extracted?
    What remains stable?
    What boundary is declared?

---

## Expected foundation behavior

A good OMNIABASE path should produce an observation artifact.

It should not silently become:

    semantic judgment
    final decision
    truth certificate
    privileged base claim

---

## Public compression

    OMNIABASE observes representation dependence.
    It does not infer meaning.
    It does not decide truth.
