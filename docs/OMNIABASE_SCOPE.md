# OMNIABASE Scope

## Purpose

OMNIABASE is a representation-aware structural observation framework.

It studies what remains stable when the same object is observed across multiple representations.

Its simplest numerical reading is:

```text
base 10 is a representation habit, not a law of number
```

## Boundary

```text
measurement != inference != decision
representation evidence != semantic truth
```

## What is in scope

OMNIABASE can study:

- multi-base numeric representation
- representation-dependent structure
- representation-resistant structure
- base-invariant properties
- notation artifacts
- structural behavior under recoding
- observer privilege caused by a single representation
- transformation-stable patterns

## What is out of scope

OMNIABASE does not certify:

- semantic truth
- mystical number meaning
- universal mathematical proof by itself
- final scientific correctness
- operational decisions
- replacement of standard arithmetic

## Correct interpretation

```text
stable across representations = structural evidence
visible only in one representation = representation-dependent evidence
neither result is final truth
```
